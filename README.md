This project will be using Node.js to allow users to watch stocks that they manually add onto their profile with news regarding those companies appearing in the news tab. Other features include the following; add stocks using tickers or companies' names, recieving real time messages when stocks drop or increase by 1.5%, have Claude also include their own sentiment if it is worth selling or buying by reading the latest news. In the initial developments of this project, we will be implementing APIs like Claude to deliver their sentiment, Finnhub to get the latest news on stocks, and AWS to handle functions such as getting and saving the watch lists and hosting the project.
